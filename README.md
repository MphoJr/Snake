# Snake Game 🐍

A classic Snake game implemented in **Python** using the **Tkinter** library for the graphical user interface.  
Players control the snake with arrow keys, eat food to grow longer, and try to avoid collisions with the walls or themselves.

---

## 🎮 Features
- Simple and responsive **Tkinter UI**.
- Snake movement controlled by **arrow keys**.
- Randomly generated food items.
- Score tracking displayed on the screen.
- Game over detection when the snake hits the wall or itself.

---

## 🛠️ Technologies Used
- **Python 3**
- **Tkinter** (GUI library)
- **Random** module (for food placement)

---

## ▶️ How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/Mphojr/Snake.git
   
   python snake.py
```  Gameplay Instructions
- Use the Arrow Keys to move the snake:
- ⬆️ Up
- ⬇️ Down
- ⬅️ Left
- ➡️ Right
- Eat the red food squares to grow longer.
- Avoid hitting the walls or your own tail.
- The score increases with each food item eaten.
```
